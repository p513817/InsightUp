# 使用說明

## 主要流程

1. 使用 Google 登入。
2. 進入 Dashboard 檢視整體趨勢圖。
3. 新增一筆 InBody 記錄。
4. 視需要切換到 Profile 頁管理記錄與 inclusion 狀態。

## Dashboard

Dashboard 提供：

- 單一主圖表
- Overall 與各部位視角切換
- 指標顯示開關
- 最新紀錄與摘要卡片

## Profile

Profile 頁面提供：

- 帳號資訊摘要
- 全部記錄管理
- inclusion 開關
- 編輯與刪除操作

## Included / Excluded 的差別

- Included：會進入圖表分析
- Excluded：只保留在歷史紀錄，不影響圖表

這是 InsightUp 的核心規則，因為 InBody 測量並不總是可靠。
